Who can use this tool?

PMM and PSP are eligible to run this tool. PMM representative for PCBA site runs PCBA allocation for the whole site every Monday for CM execution; PSP can rerun the allocation for respective BU during other days of the week upon needs of change.

What data are needed to run the tool?

  1. 3a4 Backlog: download from UOV with view: kw_PA (default includes all DF sites and all BUs, PSP can limit to certain BUs if only do allocation for certain BUs);
  2. Supply data file (includes SCR, in-transit, DF OH, sourcing rules): download from "Datasource" tab via "Download supply data file from SCDx-POC)";
  3. Exceptional priority(optional): maintained in smartsheet by PSP as needed;
  4. TAN grouping(optional): maintained in smartsheet by PSP as needed;
  5. Missing in-transit (for SRG only): maintained in smartsheet as needed by Mandy as short term fix of SCDx missing data for cross partner intransit.

How PMM run the tool on Monday for whole site?

  1. Download 3a4 and supply data file;
  2. Upload the 2 files from the "Allocation" tab;
  3. Put in appropriate PCBA org;
  4. Click "Make Allocation".
  5. Download/Share the result from the "Download" tab.

How PSP run the tool for responsible BU on other days?

  1. Download 3a4 and supply data file;
  2. Adjust the supply data file as needed - if updated supply not reflected in SCDx;
  3. Upload the 2 files from the "Allocation" tab;
  4. Put in appropriate PCBA org and BU;
  5. Click "Make Allocation".
  6. Download/Share the result from the "Download" tab.

What is order ranking priority?

We follow service 17 guideline to rank the orders. This is the priority sequence we used: PR1/2/3/L4/BUP/OSSD/YE/QTY/REVENUE_NON_REVENUE.

What is Exceptional priority?

PSP can give an exceptional priority to a non-priority SS so they can also get an absolute high priority despite OSSD and reflect in allocation - this should be used only exceptionally. Default priority ranking#: PR1:1; PR2:2; PR3:3; L4:4; BUP:5. PSP can set exceptional priority via this smartsheet. E.g., if one SS is added with Ranking# 4.1, this SS will be ranked after L4; or if added with Ranking# greater than 5, they will be after BUP; etc.

What is Tan grouping?

Some BU have different TAN supply to different DF sites but which share the same materials at PCBA level, by grouping them we can not only decide the allocation between DF but also let PCBA site know how to allocate the materials between the different TANs to best optimize the production. PSP can set the grouping as needed via this smartsheet.
